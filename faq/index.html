[PGA]

PGA is a browser extension that helps Pixels gamers play more efficiently and have more fun. PGA dramatically improves many of the frustrations of playing the Pixels game, making them much easier to play and helping you achieve the most in the least amount of time without using a bot.

Yes, it is safe to use PGA. It does not submit any actions to the Pixels game by itself. It only shows the game data that are already available inside the game. Within a month since the release, PGA has reached over 25,000 daily active users. The current user rating on the Crome store is 4.8 out of 5.

Still waiting. For mobile devices, we are making a mobile app with the same features.

PGA currently supports desktop browsers including Chrome, Edge, Brave, and Opera(GX). We are working on supporting more different browsers in the future.
