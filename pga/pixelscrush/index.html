What is the Pixels Crush game?

Pixels Crush is a match-three game where you line up three or more identical pictures.
Featuring items you harvest and produce in the Pixels game. As you progress, the score requirement increases with each level, and random empty spaces add a new challenge. Each round costs 10 Ruby Points to play.

Embrace the challenge and have fun!

Where can you play Pixels Crush game?

  • Sauna

  • Basement of each NFT land house

What is Ruby?

Ruby is an in-app point used in Pixels Game Assistant.
It is not currently a blockchain-based token and is only used in the PGA app.
Ruby points are used mainly for mini-games and other premium services offered by PGA.
